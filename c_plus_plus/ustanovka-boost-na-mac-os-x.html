    Установка Boost на Mac OS X

    less than 1 minute read

     Руки наконец дошли до boost. Самый простой способ поставить boost, на мой взгляд, это через mac ports, выполнив команду:

    sudo port install boost

    После установки все заголовки можно найти в /opt/local/include/boost

    После чего, открываем XCode, в Build Settings находим секцию Search Paths и прописываем в Header Search Paths значение /opt/local/include с опцией non-recursive. Теперь можно использовать всю мощь библиотеки. Подключить файл можно следующим образом:

    #include <boost/array.hpp>

    Кстати, в моем случае (при использовании cocos2d-x) еще пришлось добавлять эту строчку последней, чтобы не было конфликтов с библиотеками самого кокоса
